# 笔记本

arctern RESTful server 笔记本支持以下操作：列举、创建、获取、删除、克隆、运行、导出、导入。

## 请求说明

> **注意：** arctern RESTful server 的笔记本相关的请求标准与 Zepplin 保持一致。以下内容仅展示与创建、删除相关的请求示例，更详尽的使用方法请参考 <https://zeppelin.apache.org/docs/0.9.0-SNAPSHOT/usage/rest_api/notebook.html>
